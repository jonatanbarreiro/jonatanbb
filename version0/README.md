# jonatanbb.xyz

My personal site: a small, static CV / profile page — and a challenge I set myself
to build one properly, from scratch.


## Why it exists

Due to not having a defined professional dedication as I am trying to depart from
a career in academia to a place in Industry, I never got any real use out of
LinkedIn, so instead of just having a profile there I decided to make my own: a
clean, fast page to point to, and a way to stay findable for work.


## The challenge

Take it from a bare `hello world` HTML file to something I am happy to ship — in a
couple of focused weeks, AI-assisted but hand-driven. A real part of the exercise
was the AI side itself: improving on precise prompt creation, and to keep an
AI-assisted workflow organized and accountable rather than letting it sprawl.


## How it is built

- A plain static site — hand-written HTML, CSS, and JavaScript, no framework. Fast,
  legible, and cheap to host.
- Built in steps, in the open: `gallery/` holds a screenshot of the site at each
  step, so you can watch it come together at a glance; and every iteration of the
  source is kept under `sources/`.
- Served as a static site on Cloudflare.


## What it took

End to end, a project like this meant getting hands-on at once with front-end and
web-design basics, prompt-writing and AI-workflow management, everyday git and vim,
and the essentials of hosting and deploying a static site.
