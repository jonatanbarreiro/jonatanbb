The milestones of jonatanbb.xyz as a running checklist. Each line names the
iteration where the feature first became available; a checked box means it is in the
site today, an empty one means it is still ahead. When every box is checked, the site
has reached what it was built for: a clean, professional profile that makes me
visible on the job market.
